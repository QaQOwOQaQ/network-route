# 任务规划算法对比验证仿真平台

## 一、需求

### 1.1 功能点
- a）该平台用于对已有的动态路由网络建模仿真方法、网络性能预测方法、智能任务规划算法的性能进行对比验证与仿真。
- b）包括算法对比验证模块和路由网络动态仿真模块2个部分。

### 1.2 技术指标
- a）网络性能预测算法（2种），含排队论，MLP方法；
- b）节点优化排序算法（2种），含强化学习，SDN 架构下的Dijkstra 算法；
- c）能力底图构建算法（2种），含时间演进图 (Time-Expanded Graph, TEG)/接触图路由 (CGR) 建模，空间网格索引/分区建模 (Spatial Hashing/Partitioning)；
- d）路径规划算法（3种），含接触图路由 (Contact Graph Routing, CGR)，Dijkstra 算法或A*算法，分布式路由算法 (Distributed Routing)。

---

## 二、技术方案

针对每类算法，包含4个要素：
1. 场景想定、数据生成；
2. 各个子算法的实现
3. 对算法性能指标进行评估
4. 对结果可视化。
